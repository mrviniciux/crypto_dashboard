# Crypto Dashboard


Requirements: 

 * GIT
 * YARN

<b>Developer mode:</b>

 1- git clone <br>
 2- cd api_base <br>
 3- yarn <br>
 4- `yarn sequelize:database1:drop && yarn sequelize:database1:create && yarn sequelize:database1:migrate && yarn sequelize:database1:seed:all` <br>
 5- yarn dev

Starts on localhost:5000
